# OVERVIEW

	- 大衛．恩里奇所著的《黑暗巨塔：德意志銀行、唐納．川普和一場史詩般的毀滅之路》一書，詳盡且經過縝密研究，記錄了德意志銀行動盪的歷史，特別是其與唐納．川普的關係，以及在許多金融醜聞中所扮演的角色。本書全面揭露了該銀行魯莽且不道德的行為，對全球金融體系產生了深遠的影響。

# SECTION SUMMARIES

	# 序曲 [03/47] 

		這是一份關於德意志銀行及其與唐納·川普的關係，以及德意志銀行高級主管比爾·布魯克斯米特自殺事件的延伸摘要。本書講述了德意志銀行的興衰故事，以及其對世界的影響。

	# 第一章　犯罪企業 [05/47] 

		本文深入探討德意志銀行的動盪歷史，從 1870 年創立到在納粹政權中扮演的爭議角色，以及最終崛起為全球金融巨頭。敘述圍繞著兩位關鍵人物：雄心勃勃的美國鐵路大亨亨利·維拉德，以及領導德意志銀行國際擴張的魅力十足且有遠見的銀行家阿爾弗雷德·赫爾豪森。故事探討了企業行為的複雜性、權力與金融的相互作用，以及野心不受約束的後果。

	# 第二章　雙雄聯手 [06/47] 

		在華爾街這個殘酷的世界裡，兩位年輕人艾迪·米契爾和比爾·布魯克斯米特踏上了一段大膽的旅程，他們試圖透過衍生性金融商品的神秘世界來改變金融版圖。艾迪是一位積極進取、敢於冒險的業務員，他為美林證券構思了一個宏偉的策略，而比爾是一位才華洋溢但神秘的衍生性金融商品專家，他具備駕馭複雜金融工具的非凡能力。他們聯手建立了一個不太可能的夥伴關係，突破了金融的界限，積累了前所未有的財富，但他們的行為最終導致了意想不到的後果和道德困境。

	# 第三章　華爾街的大遷徒 [07/47] 

		在一位關鍵高管遭暗殺後，德意志銀行(Deutsche Bank) 旨在擴展其在德國以外的業務。該銀行的未來在於投資銀行業務，由魅力十足的米切爾(Mitchell) 領導。米切爾從他之前任職的美林證券(Merrill Lynch) 組建了一支由才華洋溢的個人組成的團隊，引發了人才大規模遷移至德意志銀行。該銀行在一年半內雇用了兩千多名員工，改變了其形象，並成為華爾街的主要參與者。

	# 第四章　暗黑勢力 [08/47] 

		美國投資銀行家進入德意志銀行，導致文化衝突和新進美國人與既有德國管理階層之間的權力鬥爭。由葛瑞格·米契爾領導的美國人，目標是現代化銀行並與華爾街巨頭競爭。另一方面，德國人抗拒改變，不信任美國人的方法。米契爾和他的團隊面臨銀行監事會和執行委員會的重大反對，他們稱之為「黑暗勢力」。儘管有這些挑戰，米契爾和他的團隊設法實施他們的變革，將德意志銀行轉變為全球衍生性金融商品市場的主要參與者。然而，他們激進的策略和缺乏風險管理最終導致德意志銀行的垮台。

	# 第五章　魚鷹計畫 [09/47] 

		1998 年，德意志銀行啟動了魚鷹計畫，一項收購華爾街主要投資銀行的雄心勃勃任務。儘管有人警告不要收購花旗集團，因為它存在財務和道德問題，但德意志銀行還是繼續收購了銀行家信託，此舉將顯著轉變銀行對投資銀行和衍生性金融商品的關注。這項收購被欺騙行為所玷污，德意志銀行也遲遲才承認自己在資助大屠殺中所扮演的角色。

	# 第六章　川普的御用銀行家 [10/47] 

		這份延伸摘要敘述了德意志銀行與唐納·川普不幸的關係，始於該銀行收購銀行信託公司——一個導致魯莽行為的重大錯誤。麥克·奧菲特，一位過去曾陷入困境的交易員，發現自己身處德意志銀行，在那裡，他和來自加州的房地產神童賈斯汀·甘迺迪在巨大的壓力下茁壯成長。儘管奧菲特有健康問題，他仍發揮了關鍵作用，建立了一個貸款證券化機器。然而，該銀行願意貸款給風險項目，吸引了聲名狼藉的川普，他過去有違約和欺騙商業夥伴的歷史。德意志銀行決定為華爾街 40 號的再融資提供給川普 1.25 億美元的貸款，這標誌著一段有問題的關係的開始，最終將損害該銀行。

	# 第七章　洪流滾滾 [11/47] 

		2000 年，艾德森·米契爾接管德意志銀行的投資銀行部門，並在員工中建立了一種恐懼和恐嚇的文化。米契爾的野心驅使他破壞合併、威脅辭職，並在企業帆船比賽中作弊。然而，當該銀行的德國母公司宣布與德累斯頓銀行合併時，米契爾團結了他的團隊並成功阻止了這筆交易。這個關鍵時刻標誌著對銀行首席執行官的尊重喪失和米契爾在組織內部權力的崛起。儘管取得了成功，但米契爾的不安全感和焦慮導致他最終垮台，當時他尋求在美林證券公司謀職。

	# 第八章　末日已近 [12/47] 

		2000 年 12 月 21 日，德意志銀行執行長艾德森·米契爾(Edson Mitchell)在一次飛機失事中突然身亡，這場意外造成權力真空，並引發組織內部的權力鬥爭。米契爾不懈追求成長和利潤，為銀行的自我毀滅奠定了基礎，而他的缺席加劇了內部衝突和管理不善，最終導致十年後的金融危機。

	# 第九章　艾克曼 [13/47] 

		約瑟夫·阿克曼，一位出生於瑞士的銀行家，於 2002 年至 2012 年領導德意志銀行。阿克曼希望德意志銀行實現 25% 的股東權益回報率 (ROE)，這導致銀行專注於短期利潤最大化，而犧牲了長期穩定性。此策略導致德意志銀行參與風險業務行為，包括破壞制裁、洗錢和過度槓桿化。阿克曼的領導風格強調個人成就勝於團隊合作和風險管理，進一步導致銀行衰落。最終，德意志銀行在阿克曼的領導下魯莽追求利潤，損害了其聲譽和財務狀況，導致 2012 年罰款 72 億美元和政府救助。

	# 第十章　海湖莊園大獎 [14/47] 

		德意志銀行為了擴展其在美國的業務，將唐納·川普視為提升其地位的機會。儘管有許多警訊，德意志銀行仍持續借給川普大量資金，使他得以開發像芝加哥川普大樓等備受矚目的專案。然而，該銀行與川普的關係證明是一把雙面刃，因為川普的財務困境和有爭議的商業行為最終損害了該銀行的聲譽。

	# 第十一章　印地安人 [15/47] 

		這段摘錄深入探討了德意志銀行在安舒·賈恩的領導下，如何屈服於過度冒險、野心勃勃和可疑的經營文化，最終導致其財務崩潰。

	# 第十二章　救火隊員 [16/47] 

		2007 年，當金融危機迫在眉睫時，德意志銀行總部發生火災，並面臨一場金融風暴。在執行長 Josef Ackermann 和 Anshu Jain 的領導下，該銀行透過大膽的決策度過了危機，包括出售風險資產和隱藏損失。儘管德意志銀行成為贏家，但其行為與其他投資銀行一樣，透過增加對風險抵押擔保證券的需求，加劇了危機。儘管避免了納稅人紓困，但德意志銀行仍因其複雜的金融押注和資產價值缺乏透明度而面臨持續的挑戰。

	# 第十三章　這個傢伙是危險人物 [17/47] 

		2008 年金融危機讓許多人負債累累，包括唐納·川普。他欠德國銀行 3.34 億美元，這筆貸款在 2008 年 11 月到期。然而，川普無意償還貸款，並指示他的律師想辦法讓他脫身。律師們注意到貸款協議包含不可抗力條款，這表示在發生不可預見的災難（例如自然災害）時，無法履行合約。他們辯稱金融危機是天災，因此川普不必償還貸款。川普援引不可抗力條款起訴德國銀行，並指控銀行從事「掠奪性貸款行為」，導致金融危機。德國銀行反訴，要求川普在 2005 年親自保證的 4000 萬美元。銀行還指出，川普曾吹噓自己如何毫髮無傷地度過危機。最後，德國銀行與川普斷絕關係，稱他為「有毒」。

	# 第十四章　風水輪流轉 [18/47] 

		德意志銀行曾經是投資者的寵兒，但在金融危機後，其資本市場和風險投資卻對其不利。監管機構和投資者擔心其資本儲備低，且大量曝險於衍生性金融商品。儘管該銀行相對穩健，但執行長約瑟夫·阿克曼的決策，例如收購郵政銀行和忽視內部警告，導致錯失機會並增加審查。該銀行的侵略和冒險文化，曾經是其優勢，但在政府監督的新時代卻成為其敗因，導致醜聞和財務困境。

	# 第十六章　羅絲瑪麗．傅瑞布麗 [20/47] 

		Rosemary Vrablic，前德意志銀行私人銀行業務主管，在她任期的最後幾個月批准了三筆高風險貸款，這些貸款後來讓銀行陷入困境。 Vrablic 以其成功和兩極分化的個性而聞名，她說服德意志銀行在 2011 年向唐納德·川普貸款 4500 萬美元，並在 2012 年為他收購 Doral 度假村和水療中心再貸款 1.25 億美元，儘管投資銀行部門提出反對意見。這些貸款最終由當時的執行長 Josef Ackermann 批准，由於川普的聲譽和違約歷史，這些貸款將成為德意志銀行的負債。

	# 第十七章　詹恩高升 [21/47] 

		2012 年，安舒·賈恩成為德意志銀行第一位非白人、非歐洲籍的執行長，這是一件大事，因為德意志銀行股東們對嚴重的金融危機仍記憶猶新。賈恩的崛起充滿爭議，因為他缺乏德語能力，而且有些人認為他沒有資格擔任這個職位。德意志銀行投資銀行的不當行為指控和賈恩在這些爭議中的角色，進一步加劇了對他領導能力的批評。

	# 第十八章　垃圾場 [22/47] 

		2012 年底，德意志銀行信託公司美洲分公司 (DBTCA) 面臨嚴峻挑戰，包括法規不足、缺乏財務資源和技術系統過時。作為德意志銀行在美國的主要營運部門，這對整個銀行構成重大威脅，可能導致其被排除在美國市場之外。DBTCA 的執行長沃爾夫辭職，傑克·布蘭德接任，並聘請布羅克史密斯協助解決問題。布羅克史密斯找出系統性問題，並協助實施變革以改善合規性和財務報告。在這些挑戰中，布羅克史密斯與米契爾的兒子史考特重新聯繫，兩人言歸於好，恢復了舊有的家庭關係。

	# 第十九章　索取資訊 [23/47] 

		在安舒·賈恩(Anshu Jain)的領導下，德意志銀行面臨一系列調查和法律問題，包括逃稅、洗錢和操縱倫敦銀行同業拆借利率(Libor)的調查。儘管監管機構發出警告，但該銀行仍繼續從事風險行為，包括協助俄羅斯客戶通過德意志銀行洗錢的鏡像交易計畫。

	# 第二十章　壓力重重 [24/47] 

		在這本引人入勝的敘述中，我們深入探討了銀行業高風險世界中，企業高管所承受的無情壓力。故事透過布魯克斯·米特爾斯通的視角展開，他是一位經驗豐富的華爾街高管，正努力應對董事會提名失敗的後果，以及他之前公司德意志銀行所面臨的挑戰。米特爾斯通希望與銀行日益嚴重的問題保持距離，但這種願望與他保護銀行聲譽的責任感發生了衝突。當他應對退休、人際關係的複雜性，以及他在德意志銀行內部與道德困境的抗爭時，米特爾斯通的旅程提供了一個發人深省的提醒，即在殘酷的金融世界中追求成功所付出的個人代價。

	# 第二十一章　華倫亭 [26/47] 

		在 1970 年代烏克蘭的政治壓迫中，艾拉·切列德尼琴科和她的家人移民到美國，在那裡，她的兒子瓦爾在父母離異後被寄養。儘管被一個充滿愛的家庭收養，瓦爾仍與情緒動盪和行為問題作鬥爭。在青少年時期，他在音樂中找到了慰藉，並組建了一個名為 Charlie Goodtimes 的樂隊。儘管他在音樂產業取得了一些成功，但瓦爾仍繼續與被遺棄和自我懷疑的感覺作鬥爭。與一位名叫貝絲的威爾斯女性的偶然相遇為他的生活帶來了穩定，但未解決的童年創傷仍然存在。

	# 第二十二章　生命結束 [27/47] 

		46 歲的布魯克斯密斯患有憂鬱症，在一個雨天的星期日上午在家中用狗鍊上吊自殺。他的妻子艾拉回到家後發現他，並嘗試進行心肺復甦術，但無法救活他。醫護人員在下午 1:00 宣布他死亡。

	# 第二十三章　天翻地覆 [28/47] 

		本章深入探討德意志銀行(Deutsche Bank)一位傑出主管比爾·布魯克斯米特(Bill Broeksmit)自殺後的後續影響。當他的家人和同事試圖理解他突然死亡的原因時，他們發現了一些線索，表明布魯克斯米特可能隱藏著一些秘密，這些秘密可能導致他的絕望。隨著媒體的關注和警方的介入，布魯克斯米特行為背後的真相仍然難以捉摸，讓與他最親近的人陷入複雜的情緒和未解的問題中。

	# 第二十四章　不必擔心 [29/47] 

		儘管希臘銀行 Hellenic 擁有良好的財務法規，但它發現大量資金存入俄羅斯人擁有的帳戶中，這引起了懷疑。他們發現資金來源是德意志銀行，而德意志銀行未能提供令人滿意的解釋，這促使進一步調查，最終揭露了德意志銀行經營的洗錢計畫。在金融危機的漩渦中，德意志銀行發現自己處於公眾和政府的嚴密審查之下，內部鬥爭和緊張的關係加劇了這種情況。儘管試圖復甦，但該銀行面臨著許多法律和財務挑戰，揭露了投資銀行業內的不當行為和監管失敗的文化。

	# 第二十五章　可憐的天才布羅克斯密特 [30/47] 

		《華爾街日報》記者葛瑞格·祖克曼和他的團隊揭露了一則關於德意志銀行一位高級主管自殺的故事。該團隊在紐約聯邦儲備銀行的一封信中強調了銀行的「嚴重問題」，包括財務報告不佳，揭露了這位主管對銀行的沮喪。這封信的發現導致德意志銀行股價下跌，並引發了對該銀行業務的調查。

	# 第二十六章　北韓之亂 [31/47] 

		2014 年，在德意志銀行一名律師自殺後，沃爾展開了一項針對其父親死亡事件的個人調查。他發現一個神秘的黑客組織「和平守護者」入侵了索尼影業娛樂公司。沃爾聯繫了這些黑客，並獲得了被盜索尼檔案的訪問權，隨後他將這些檔案洩露給了公眾。這些洩露揭露了索尼的內部溝通，並曝光了企業的不當行為。沃爾的行為最終導致路透社發表了一篇揭露德意志銀行欺騙行為的文章。

	# 第二十七章　信心缺缺 [32/47]

	# 第三十一章　西恩納 [36/47] 

		全球銀行巨頭德意志銀行在 2016 年面臨嚴峻的財務和法律挑戰。該銀行被指控隱瞞損失並面臨不當行為調查，其償付能力受到質疑。在混亂中，亞歷克斯·羅斯·瓜爾蒂耶里（Alex Ross Gualtieri）踏上了一項個人使命，試圖揭開其父親死亡背後的事實，以及德意志銀行的困境。瓜爾蒂耶里在其他人的幫助下，發現了德意志銀行風險衍生性商品交易、義大利蒙特帕斯基迪錫耶納銀行的金融崩潰以及潛在犯罪活動之間的關聯。他的調查揭示了德意志銀行不懈追求利潤和無視道德行為的深遠後果。

	# 第三十三章　別說出「川普」這個名字 [38/47]

	# 第三十五章　總統來函 [40/47]
